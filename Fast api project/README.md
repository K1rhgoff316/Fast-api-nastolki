# Клуб настольных игр API

Простой REST-сервис на FastAPI. Он нужен клубу настольных игр: добавить игры, запланировать встречу, записать участников, оставить отзывы и подобрать игру под компанию.

## Как запустить

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload
```

Откройте Swagger: `http://127.0.0.1:8000/docs`.

## Как пользоваться в Swagger

1. Откройте `POST /auth/register` и создайте пользователя.
2. Откройте `POST /auth/login`, введите `username` и `password`, скопируйте `access_token`.
3. Нажмите кнопку `Authorize` справа сверху и вставьте токен.
4. Откройте `POST /games` и добавьте игру.
5. Откройте `POST /sessions` и создайте игровую встречу.
6. Откройте `POST /reviews` и оставьте оценку игре.
7. Откройте `GET /analytics/recommendations`, введите число игроков и получите рекомендацию.

Если забыли порядок действий, откройте главную страницу API: `http://127.0.0.1:8000/`.

## Что есть в проекте

- FastAPI REST API.
- SQLite-БД создается автоматически.
- 5 связанных таблиц: пользователи, игры, встречи, участники, отзывы.
- JWT/OAuth2-авторизация.
- CRUD для игр, встреч, участников и отзывов.
- Бизнес-логика: подбор игры по числу игроков, длительности, сложности и рейтингу.
- Тесты через `TestClient`.

## Главные эндпоинты

| Что сделать | Эндпоинт |
| --- | --- |
| Создать пользователя | `POST /auth/register` |
| Войти | `POST /auth/login` |
| Добавить игру | `POST /games` |
| Создать встречу | `POST /sessions` |
| Оставить отзыв | `POST /reviews` |
| Подобрать игру | `GET /analytics/recommendations` |

Всего в API 20 эндпоинтов, поэтому требования по CRUD и связанным сущностям закрыты.
